# Blinking Page

A minimalist web project that creates a rapid, multi-color strobe effect using pure HTML and CSS.

##### ⚠️ Warning: This project contains extremely fast flashing lights and color changes. It is not suitable for individuals with photosensitive epilepsy.

## Overview

This project utilizes a simple HTML structure and CSS animations to flash the entire background of the viewport. The animation cycles through 11 different colors in a continuous loop every 225 milliseconds.

## Setup & Usage

Since this is a static web project, no installation is required:
1. Clone this repository or copy the code files.
2. Ensure your file structure looks like this:
- index.html
- style.css
3. Open index.html in any modern web browser.

## Technical Details

- Animation: @keyframes blink
- Duration: 225ms (Approx. 4.4 cycles per second)
- Loop: infinite
- Stack: HTML5 & CSS3

## Color Sequence

The animation transitions through the following colors at specific keyframes:
- Yellow, White, Blue, Green, Red, Purple, Cyan (Custom Hex), Orange, Pink, and Black.
